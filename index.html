About

The HaLVM allows Haskell programmers the opportunity to compile their programs directly to virtual "bare metal." This allows them to control their environment completely, down to the level of device drivers, and can provide additional performance, security, and cost benefits. For example, HaLVMs run in vastly reduced memory and disk footprints (saving you money), without any unnecessary code or libraries (shrinking your attack profile), and without needless schedulers or competing projects (improving your performace). Futher, the strong, flexible type system allows you to prove critical properties of your program before it even launches.

Get started with HaLVM

Writing programs for the HaLVM is just like writing program for a normal system, but with its own special quirks and tweaks. In fact, we strongly encourage users to start their HaLVM project by implementing their target service on Linux, macOS, or Windows, and only worry about the HaLVM afterwards.


There already? Then peruse our tutorials for tips and tricks. Our "So you want to build a HaLVM" choose-your-own-adventure tutorial is probably the best place to start.


About Galois

The HaLVM is an open-source offering of Galois, Inc., a Research and Development firm specializing in improving the trustworthiness of critical systems. Galois developed the HaLVM as a prototyping tool to help improve the state of modern operating system design, and continues to use the HaLVM in projects that rely on secure components running in virtual environments.
